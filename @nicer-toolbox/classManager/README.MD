# Nicer Toolbox

## Class Manager

Improves className management in javascript.

### Class Manager Simple Class

```js
// For React
import { cmsc } from '@nicer-toolbox/class-manager';

// Example
const App = () => {
  return (
    <div className={cmsc(['relative', 'bg-white', 'overflow-hidden'])}></div>
  );
};
```
